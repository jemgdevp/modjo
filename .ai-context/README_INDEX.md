# Índice de `.ai-context`

Este archivo centraliza un índice y resúmenes breves de los archivos y carpetas dentro de `.ai-context` para agentes y colaboradores.

## Estructura general

- `SKILLS/` — Carpeta raíz para skills de agentes. 
- `README_INDEX.md` — Explica el propósito de la carpeta `.ai-context` y su estructura (goals/, knowledge/, rules/, tasks/). 
- `goals/` — (Vacío) carpeta destinada a metas y objetivos del proyecto. Actualmente no contiene archivos.
- `knowledge/` — Documentación técnica y de integración.

## Notas rápidas

- Las carpetas `goals/` y `tasks/` están vacías y pueden recibir ficheros `md` pequeños y enfocados (meta/tareas respectivamente).
- `knowledge/` contiene información de alto valor para desarrolladores y agentes; mantén esos archivos actualizados cuando cambie la arquitectura o se añadan integraciones.

Si quieres que indexe también otros directorios o que genere resúmenes más largos (por ejemplo, un artículo o checklist derivado de alguno de estos archivos), dímelo y lo hago a continuación.
